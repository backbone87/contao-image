=== Titel
Image processing library


=== Suchbegriffe
api,gdlib,resample,thumbnail,image


=== Teaser
Library for developers to process images.


=== Beschreibung
This library is work in progress.
Testcases, feedback and comments are welcome.

Features:

    * Generating thumbnails
    * Based on gdlib
    * Several utility functions to calculate image dimensions and aspect ratios
    * Output to file or binary string
    * Copy resampled areas between images
    * Watermark function
